# Python Backend User Manager

Simple backend-style user management system written in Python.

## Description
This project simulates common backend development patterns such as data modeling,
validation, error handling, and JSON-based data persistence, without relying on
external frameworks.

## Features
- User data model
- In-memory storage with JSON persistence
- Basic CRUD operations
- Input validation and error handling

## Goals
- Practice backend logic using Python
- Apply clean code principles
- Build a foundation for a future REST API

## Tech Stack
- Python 3.x
- Standard Library only
