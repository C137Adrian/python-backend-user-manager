# Python Backend User Manager

Simple backend-style user management system developed in Python.

## Features
- Create, read, update and delete users
- Input validation
- JSON-based persistence
- Clean and maintainable architecture
- CLI interface (backend logic focused)

## Purpose
This project was created to practice backend development concepts using Python,
including data persistence, validation, and clean code principles.

## Tech Stack
- Python 3.x
- Standard Library only
